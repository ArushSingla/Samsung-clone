A modern, responsive clone of the Samsung Galaxy S25 Ultra product page built using HTML, CSS, and JavaScript. This project focuses on replicating a premium UI/UX experience with smooth layouts and responsive design.

⸻

🚀 Features
	•	✨ Fully responsive design (mobile, tablet, desktop)
	•	🎨 Modern and premium UI styling
	•	🖼️ High-quality product layout sections
	•	⚡ Smooth scrolling and subtle animations
	•	📦 Clean and structured code
	•	🔘 Interactive elements (buttons, hover effects)

⸻

🛠️ Tech Stack
	•	HTML5 – Structure
	•	CSS3 – Styling & Animations
	•	JavaScript (Vanilla JS) – Interactivity
